# cloud-notes

Reader-friendly, all-in-one quick references for **Cloud & DevOps** — AWS services, Docker, CI/CD, Linux, scripting, and more.

🔗 **Live site:** https://ramankharchee.github.io/cloud-notes/

Each topic ships as a Markdown source (`*-notes.md`), a rendered HTML page, and a PDF.

## Interview prep

- **[Scenario-Based Interview Questions](scenario-based-notes.md)** — cross-service, multi-step scenarios (EC2, S3, RDS, Lambda, SQS/SNS, Docker, VPC, security & RBAC) with model answers and the follow-ups an interviewer pushes.
- Each service note (EC2, S3, RDS, Docker, …) also ends with a **Detailed Interview Q&A and When to Use** section.
